留言和评论

本博客是一个静态网站，没有可以处理留言和评论的服务端。作为替代，本博客使用 GitHub 仓库的 issue 区作为评论区。

对于博客的一般性问题，请在 General Comments 区提交 issue。

对于博文的评论，请点击博文结尾处提供的链接，在 issue 区的 Milestones 中寻找和博文同名的条目，新建 issue，或评论他人的 issues。
